## 🏗️ Installation & Setup

To run locally:

```sh
npm install
npm run dev
```

To build for production:

```sh
npm run build
```

To deploy:

```sh
npm run deploy
```

## 🛠️ Technologies Used

- **React**
- **TypeScript**
- **Vite**
- **GitHub Pages for Deployment**
